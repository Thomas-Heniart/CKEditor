# CKEditor

This project is a binding of CKEditor for [Seaside](https://github.com/SeasideSt/Seaside)

# Installation
## Installation in Pharo5
```smalltalk
Metacello new
	baseline: 'CKEditor';
	repository: 'github://titomheniart/CKEditor:master';
	load
```
